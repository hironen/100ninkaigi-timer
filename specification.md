# 発表タイマー (10分) 仕様書

## 1. 概要
本アプリケーションは、10分間のプレゼンテーションを支援するための高視認性Webタイマーである。離れた場所からでも残り時間と状況を直感的に把握できるよう、巨大な数字表示と画面全体の色変化、動的なメッセージ表示を備えている。

## 2. 機能仕様

### 2.1 カウントダウン機能
- 初期設定時間: 10分 (600秒)
- 制御: START / PAUSE / RESET
- 精度: 1秒単位

### 2.2 表示ロジック
残り時間に応じて、表示形式とメッセージが動的に変化する。

| 残り時間 | 表示形式 | メッセージ | 背景色 |
| :--- | :--- | :--- | :--- |
| 開始時 ～ 2:01 | 分:秒 | (空) | 黒 (#111) |
| 2:00 ～ 1:01 | 分:秒 | **そろそろまとめて** | **青 (#007bff)** |
| 1:00 ～ 0:51 | 分:秒 | **ラストスパート** | **黄 (#ffd700)** |
| 0:50 ～ 0:31 | **秒数のみ** | **ラストスパート** | **黄 (#ffd700)** |
| 0:30 ～ 0:11 | **秒数のみ** | **ラストスパート** | **赤 (#d40000)** |
| 0:10 ～ 0:01 | **秒数のみ** | **まもなく終了** | **赤/黒 点滅** |
| 0:00 (終了) | **0** | **おつかれさまでした** | 黒 (#111) |

### 2.3 終了演出 (爆発エフェクト)
- 残り時間が0秒になった瞬間に、画面中央からカラフルなパーティクルが飛び散る爆発アニメーションを実行する。
- リセットを行うことで、次回の終了時にも再度実行可能となる。

## 3. デザイン・UI仕様

### 3.1 タイポグラフィ (高視認性設計)
画面幅に応じてフォントサイズが自動調整される `clamp` 関数を採用し、巨大な表示を実現している。

- **タイマー数字 (`#time`)**: 140px ～ **500px**
- **メッセージ (`#msg`)**: 40px ～ **120px**
- **フォント**: システムフォント (Noto Sans JP 等) を優先し、ウェイト 800 (超極太) を使用。

### 3.2 レスポンシブ対応
- 画面の中央にコンテンツを配置し、デバイスの向きや解像度に関わらず視認性を維持。
- 操作ボタンは画面下部に固定し、プレゼン中の誤操作を防止。

### 3.3 インタラクション
- ボタン操作以外に、画面のどこをタップしても **START / PAUSE** の切り替えが可能。

## 4. 技術仕様
- **言語**: HTML5, CSS3, JavaScript (Vanilla JS)
- **依存関係**: なし (単一のHTMLファイルで完結)
- **パフォーマンス**: 
    - アニメーションに `requestAnimationFrame` を使用し、スムーズな描画と低負荷を実現。
    - CSS変数 (Custom Properties) による効率的な色管理。
